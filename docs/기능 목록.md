# 🚀 도메인 로직

1. 다리 생성하기
2. 이동
3. 시도횟수 관리

## 도메인 로직 기능 목록

- [] 다리 생성하기

  - [] 랜덤값 부여하기
  - [] 자료구조 활용

- [] 이동

  - [] 이동에 따른 다리의 상태 수정
  - [] 결과 출력 범위 수정

- [] 시도횟수 관리
  - [] 게임 시작을 포함한 시작횟수 저장

# 🧑‍💻 애플리케이션 서비스 로직 기능 목록

- [] 에러 발생 시, 그 부분에서 다시 시작하기

# 🚫 에러 발생

- [] 다리 길이가 `3 ~ 20` 사이가 아닌 경우
- [] 이동 입력이 `U` or `D`가 아닌 경우
- [] 재시작 입력이 `R` or `Q`가 아닌 경우
